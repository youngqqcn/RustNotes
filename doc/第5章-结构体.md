
## 第5章 结构体

结构体分类:

- 普通结构体: 包含数据字段的 
- 元组结构体: 没有字段名称, 只有字段类型
- 类单元结构体: 不包含数据字段的, 仅用来实现某些trait

为结构体实现方法

```rust
struct Rectangle {
    width: u32,
    height: u32,
}

impl Rectangle {
    fn area(&self) -> u32 {
        self.width * self.height
    }

    //关联方法, 在C++中叫静态方法
    fn name() -> String {
        String::from("Rectangle")
    }
}

impl Rectangle {
    fn get_width(&self) -> u32 {
        self.width
    }
}

fn main() {
    let rect = Rectangle { 
        width: 100,
        height: 100,
    };

    println!("面积: {}", rect.area());
    println!("name: {}", Rectangle::name());
    println!("width:{}", rect.get_width());
}
```
